# 📚 مستودع القوانين اليمنية الأساسية

هذا المستودع يحتوي على النصوص الكاملة للقوانين اليمنية الأساسية، المعدة للاستخدام في الأعمال القانونية.

## القوانين المتوفرة حالياً

1. **قانون المرافعات والتنفيذ المدني**
   - القانون رقم **(40)** لسنة **2002م**.
   - تعديلاته: القانون رقم **(2)** لسنة **2010م**، والقانون رقم **(21)** لسنة **2021م**.

2. **قانون التجارة**
   - القانون رقم **(32)** لسنة **1991م** (معدل بالقانون رقم 16 لسنة 1998م).

3. **قانون الإثبات**
   - القانون رقم **(21)** لسنة **1992م**.

## قوانين ستضاف قريباً

- القانون المدني رقم (14) لسنة 2002م.
- قانون الأحوال الشخصية.
- قانون الشركات.
